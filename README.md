# 🚀 Flui CLI - Assistente Ultra Autônomo

Sistema de CLI com IA integrada que executa tarefas de forma 100% autônoma e dinâmica.

## ✨ Características

- **100% Autônomo**: Executa tarefas complexas sem supervisão
- **Sem Templates**: Gera código dinâmico e original
- **Auto-correção**: Detecta e corrige erros automaticamente
- **Score 90%+**: Garante qualidade mínima em todas as tarefas
- **Ultra Capacidade**: Usa toda capacidade disponível para executar

## 📦 Instalação

```bash
git clone https://github.com/fera765/flui-cli
cd flui-cli
npm install
```

## 🎯 Como Usar

### Comando Único

```bash
npm run dev
# ou
npm run flui
```

Simplesmente digite sua tarefa e o Flui executará de forma autônoma!

## 💡 Exemplos de Tarefas

```bash
# Frontend
> Crie um e-commerce completo com React e Tailwind

# Backend  
> Desenvolva uma API REST com autenticação JWT

# Fullstack
> Crie um sistema completo de blog com painel admin

# Conteúdo
> Gere um artigo de 10000 palavras sobre inteligência artificial

# Mobile
> Desenvolva um app de tarefas com React Native

# CLI
> Crie uma ferramenta CLI para gerenciar projetos
```

## 🤖 Como Funciona

1. **Digite a tarefa**: O Flui analisa o que você precisa
2. **Execução Autônoma**: Flui executa todos os passos necessários
3. **Auto-correção**: Se encontrar erros, corrige automaticamente
4. **Validação**: Garante score mínimo de 90% de qualidade
5. **Entrega**: Projeto completo e funcional

## 🚀 Diferencial

O Flui CLI é diferente porque:

- **Não usa templates fixos** - Cada projeto é único
- **Não tem mocks em produção** - Código real e funcional
- **Auto-validação** - Garante qualidade automaticamente
- **Ultra autônomo** - Usa máxima capacidade disponível
