# Flui CLI

Sistema autônomo de geração de código com IA.

## Instalação
```bash
npm install
npm run build
```

## Uso
```bash
npm run dev        # Modo produção
npm run autonomous # Modo autônomo
npm run ultra      # Ultra autônomo com monitoramento
```

## Features
- 100% autônomo e dinâmico
- Auto-validação e correção
- Score mínimo 90%
- Sem templates ou mocks
- Monitoramento em tempo real
